## 1. 디자인 시스템

### 1.1 적용 스타일
**UI-UX-Pro-Max-Skill** 기반 자동 생성

- **스타일**: Modern Dashboard (Glassmorphism + Clean)
- **산업**: Tech/SaaS
- **컬러 팔레트**: Blue-based Professional
- **폰트**: Inter (본문) + Pretendard (한글)

### 1.2 컬러 팔레트

## 7. 애니메이션 & 인터랙션

### 7.1 Micro-interactions

- **버튼 호버**: scale 1.02, 150ms ease
- **카드 호버**: shadow-lg, 200ms ease
- **좋아요 버튼**: scale 1.2 → 1.0, 300ms spring
- **체크 완료**: confetti 효과 (optional)

### 7.2 Page Transitions

- **페이지 전환**: fade-in, 200ms
- **모달 오픈**: fade + slide-up, 250ms
- **사이드바 토글**: slide-in-left, 200ms

### 7.3 Loading States

- **Skeleton**: 콘텐츠 로딩 시 플레이스홀더
- **Spinner**: 액션 진행 중
- **Progress**: 파일 업로드 등

## 8. 접근성 (A11y)

### 8.1 키보드 네비게이션
- Tab 순서 논리적으로 설정
- Focus visible 스타일 적용
- Skip to main content 링크

### 8.2 스크린 리더
- 적절한 aria-label 사용
- 이미지에 alt 텍스트
- 폼 요소에 label 연결

### 8.3 색상 대비
- WCAG 2.1 AA 기준 충족 (4.5:1)
- 색상만으로 정보 전달하지 않음
