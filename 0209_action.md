# GOOTE 프로젝트 진행 내역 (2026-02-09)

## 1. 진행 완료 내역

### 🛠 기초 인프라 구축
*   **저장소 연동**: GitHub 저장소(`guyster323/GOOTE`)를 로컬 환경과 연동 완료.
*   **프론트엔드 초기화**: Next.js 14 (App Router), TypeScript, Tailwind CSS, shadcn/ui, Zustand, React Query 설정 완료.
*   **백엔드 초기화**: Firebase (Auth, Firestore, Functions, Hosting) 환경 설정 및 초기화 완료.

### 🔐 인증 및 사용자 관리
*   **Google 로그인**: Firebase Auth를 이용한 구글 로그인 기능 구현.
*   **인증 상태 관리**: Zustand를 이용한 전역 인증 상태 관리 및 `useAuth` 커스텀 훅 구현.
*   **사용자 초기화**: 신규 가입 시 Firestore에 사용자 프로필 및 초기 통계 데이터를 자동 생성하는 백엔드 트리거(`onUserCreated`) 구현.

### 📱 앱 관리 (개발자용)
*   **대시보드 레이아웃**: 사이드바 네비게이션을 포함한 대시보드 구조 구축.
*   **앱 등록 폼**: Zod와 React Hook Form을 이용한 유효성 검사 및 Firestore 연동 완료.
*   **통계 자동 업데이트**: 새로운 앱 등록 시 개발자의 등록 앱 수(`appsRegistered`)를 자동 증가시키는 트리거 구현.

### 🔍 앱 탐색 및 참여 (테스터용)
*   **Explore 마켓플레이스**: 모집 중인 앱 목록을 카테고리별로 검색 및 필터링할 수 있는 페이지 구현.
*   **테스트 참여 로직**: "참여하기" 클릭 시 참여 기록 생성 및 앱별 참여자 수 자동 집계 트리거 구현.
